## About

Explore a beautiful snowy Lapland Taiga forest! Watch the poros (reindeer) roam freely through the pine trees while snowflakes gently fall around you.

## Controls

| Key | Action |
|-----|--------|
| `W` `A` `S` `D` | Move |
| `SPACE` | Jump |
| `Mouse` | Look around |
| `Click` | Lock pointer |

## Features

- Procedurally generated snowy terrain
- Pine trees with snow-covered branches
- Animated poros (reindeer) wandering the taiga
- Falling snowflakes
- First-person movement with physics
