# RUKA-PORRO

A 3D Voxel Game set in the magical Lapland Taiga

## Play Now

**[Play RUKA-PORRO](https://maca2024.github.io/RUKA-PORRO/)**

## About

Guide a cute voxel-style Poro (Finnish reindeer) through a snowy Lapland forest under the northern lights. Collect glowing lichens while enjoying the peaceful winter atmosphere!

## Controls

| Key | Action |
|-----|--------|
| `W` / `Arrow Up` | Move forward |
| `S` / `Arrow Down` | Move backward |
| `A` / `Arrow Left` | Move left |
| `D` / `Arrow Right` | Move right |
| `Space` | Jump |
| `Mouse` | Look around (click to enable) |

## Features

- Voxel-style graphics
- Procedurally generated pine forest
- Falling snow particles
- Northern lights atmosphere
- Collectible glowing lichens
- Third-person camera controls

## Running Locally

Simply open `index.html` in a modern web browser (Chrome, Firefox, Edge, Safari).

## Technology

Built with:
- [Three.js](https://threejs.org/) - 3D graphics library
- Vanilla JavaScript
- HTML5/CSS3
